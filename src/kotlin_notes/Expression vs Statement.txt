* Expressions:
	Expressions are pieces of code that have a specific value.
	
	Examples of some expression: 
			1> 	3
			2>	3+6
			3>	listOf(1,2,3,4)
			4>	"Kotlin" + "> Java"			
			
	Expression have type also, in above examples.
		1> Integer
		2> Integer
		3>  In this the return type is the type of expression i.e. Integer
		4> String
